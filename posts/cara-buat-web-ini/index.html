Kenapa Bikin Web Ini?#

Sebenernya sih… nggak ada alasan yang terlalu keren.
Cuma pengen punya tempat buat nulis sambat, random, dan kadang-kadang sok produktif.

Stack yang Dipakai#

  • Hugo → Static Site Generator (SSG) super cepat.
  • PaperMod → Tema minimalis, clean, dan gampang dikustomisasi.
  • Git → Buat version control.
  • GitHub Pages → Hosting gratis untuk deploy.
  • PowerShell / Terminal → Jalankan perintah.
  • VS Code → Edit file dan konfigurasi.

Step-by-step (Biar Nggak Lupa)#

  1. Install Hugo

    • Download & install via Chocolatey biar gampang.
    • Perintahnya:
      choco install hugo-extended

  2. Buat Folder Project

    hugo new site chospace
